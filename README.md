## What I Build

Systems programmer. My main project is **[Perdition-OS]( https://github.com/RPDevJesco/tutorial-os-definitive-edition )** (formerly Tutorial-OS). A from-scratch, multi-architecture operating system targeting x86_64 (LattePanda IOTA/Mu), ARM64 (Raspberry Pi 5), and RISC-V (Orange Pi RV2). C owns the silicon covering boot, MMU, cache, MMIO. Rust owns the data covering device implementations, runtime modules, everything above the FFI boundary. The split is enforced by architecture, not convention.
It runs on real hardware, not just QEMU. Current state includes a graphical launcher, an ELF loader, a work-stealing scheduler, a bare-metal Game Boy emulator, and FBX/glTF parsing with zero dependencies.
